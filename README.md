# ExperimentsCloud

ExperimentsCloud is a lightweight personal cloud storage application built with Flask.

## Features

- User authentication
- Secure file upload
- File download
- File deletion
- UUID-based file storage
- Centralized configuration
- Centralized logging
- Error handling
- Environment-based configuration

## Tech Stack

- Python
- Flask
- SQLAlchemy
- Flask-Login
- Flask-Migrate
- SQLite
