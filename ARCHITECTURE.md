## UI Foundation

ExperimentsCloud uses reusable UI components.

Current components:

- Button System
- Table System
- Badge System
- Utility Classes

Future components:

- Navbar
- Pagination
- Modal
- Dropdown

## Template Components

Reusable UI components are stored in:

- templates/components/navbar.html
- templates/components/flash.html
- templates/components/footer.html

The base template only assembles components and provides content blocks.
